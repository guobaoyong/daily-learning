## Build Setup


```bash
# clone the project
git clone https://github.com/xugaoyi/wx-music-admin-backend.git

# enter the project directory
cd wx-music-admin-backend

# install dependency
npm install

# develop
npm run dev
```

 访问地址：http://localhost:3000
